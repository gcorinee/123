# 칸타타 Cantata
## Abstraction
약속을 편하게 잡자!
서로의 일정을 확인해 약속을 편하게 잡을 수 있다.
- 친구의 프로필을 통해 정보 확인, 캘린더 조회, 약속 요청 가능
- 친구의 캘린더를 통해 친구의 월, 일별 약속을 확인 할 수 있다.
- 게시판을 통해 소통 가능
## 개발 환경
### Front-end
- OS: Android
- Android Studio
- Kotlin
### Back-end
- 서버: nodejs
- 데이터베이스: mongo DB
